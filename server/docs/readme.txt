项目结构，MTV模式

|---- server.py        # 启动服务
|---- app              # 存放MTV应用包
|-------- __init__.py  # 初始化模块，自定义应用，自定义服务
|-------- configs.py   # 配置信息（站点，数据库）
|-------- urls.py      # 路由试图映射配置文件（配置路由规则）
|-------- views        # 视图包（handlers）
|-------- models       # 模型包
|-------- templates    # html模版目录
|-------- static       # 静态文件目录（css, js, 图片）
|-------- tools        # 工具包（图表，获取硬件信，连接数据库）
